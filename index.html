Changelog

v1.0.1
📐 Collapsible Calendar
Hide the entire calendar grid with one click. The date inputs and navigation remain visible. Toggle collapses/expands the calendar body.
New Feature
v1.0.1
📏 Date Input Text Size Control
New setting in the Calendar card to independently control the font size of date input boxes. Responsive scaling with a maximum limit to prevent oversized inputs.
New Feature
v1.0.1
🔧 Improved Date Input Responsiveness
Date input boxes now use min-width/max-width instead of fixed 'ch' units. Text overflow is handled with ellipsis. Max 10 characters limit added.
Improvement

Privacy Policy

Last Updated: July 2026

Persian Calendar Pro does not collect, store, process, transmit, or share any personal or business data. The visual operates entirely within Microsoft Power BI and only uses the data provided by the report. No information is sent to external servers. No cookies, tracking, analytics, or advertising services are used.
