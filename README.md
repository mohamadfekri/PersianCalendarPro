# Persian Calendar Pro

Professional Persian (Jalali) Calendar custom visual for Microsoft Power BI.

## Features

- Full Jalali (Persian) calendar support
- Responsive layout
- Modern and clean user interface
- Configurable colors and appearance
- Optimized performance
- Compatible with Microsoft Power BI

## Installation

1. Download the `.pbiviz` package.
2. Open Microsoft Power BI Desktop.
3. Import the custom visual.
4. Add the visual to your report.

## Usage

Drag the Persian Calendar Pro visual onto the report canvas and configure it using the available formatting options.

## Requirements

- Microsoft Power BI Desktop
- API Version 5.11.0 or later
